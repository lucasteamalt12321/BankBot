def sort_names_by_grade(data):
  """
  Принимает кортеж пар (имя, оценка) и возвращает кортеж имен,
  отсортированных по убыванию оценок. Сохраняет порядок для одинаковых оценок.

  Args:
    data: Кортеж кортежей вида (имя_строка, оценка_целое_число).
          Пример: (('Аня', 88), ('Борис', 91), ('Виктор', 88))

  Returns:
    Кортеж строк (имен), отсортированных по оценкам в убывающем порядке.
    Пример: ('Борис', 'Аня', 'Виктор')
  """
  # 1. Сортируем исходный кортеж.
  #    - `sorted()` создает *новый* отсортированный *список*.
  #    - `key=lambda item: item[1]` указывает, что сортировать нужно по второму элементу
  #      каждого вложенного кортежа (т.е. по оценке).
  #    - `reverse=True` задает сортировку по убыванию (от большего к меньшему).
  #    - Важно: `sorted()` в Python является *стабильной* сортировкой. Это значит,
  #      что если два элемента имеют одинаковый ключ (оценку), их относительный
  #      порядок в отсортированном результате будет таким же, как и в исходном
  #      кортеже. Это как раз то, что нам нужно для условия "сохраняется порядок появления".
  sorted_list = sorted(data, key=lambda item: item[1], reverse=True)

  # 2. Извлекаем только имена из отсортированного списка кортежей.
  #    - Используем списковое включение (list comprehension) для краткости.
  #    - `item[0]` берет первый элемент (имя) из каждого кортежа `item`.
  sorted_names = [item[0] for item in sorted_list]

  # 3. Преобразуем полученный список имен обратно в кортеж.
  result_tuple = tuple(sorted_names)

  return result_tuple

# Пример использования:
input_data = (('Аня', 88), ('Борис', 91), ('Виктор', 88), ('Галина', 95), ('Дмитрий', 91))
output_data = sort_names_by_grade(input_data)

Пояснение кода:

1. `sorted(data, key=lambda item: item[1], reverse=True)`:
* data: Исходный кортеж, который мы передаем в функцию.
* key=lambda item: item[1]: Это самая важная часть для сортировки. lambda создает маленькую анонимную функцию. Эта функция берет один элемент (item) из data (который является кортежем типа ('Имя', Оценка)) и возвращает item[1] (то есть оценку). Функция sorted использует эти возвращенные значения (оценки) для сравнения элементов.
* reverse=True: Указывает sorted сортировать в порядке убывания (от наибольшей оценки к наименьшей).
* Результатом sorted() будет *список* кортежей, отсортированных по оценкам. Например: [('Галина', 95), ('Борис', 91), ('Дмитрий', 91), ('Аня', 88), ('Виктор', 88)]. Обратите внимание, что 'Борис' идет перед 'Дмитрием', а 'Аня' перед 'Виктором', так как они были в таком же порядке в исходных данных, и сортировка стабильна.

2. `[item[0] for item in sorted_list]`:
* Это списковое включение (list comprehension). Оно проходит по каждому item (кортежу) в sorted_list.
* Для каждого item оно берет item[0] (имя).
* В результате создается *список* только из имен в нужном порядке: ['Галина', 'Борис', 'Дмитрий', 'Аня', 'Виктор'].

3. `tuple(sorted_names)`:
* Преобразует полученный список имен (sorted_names) в кортеж.
* Результат: ('Галина', 'Борис', 'Дмитрий', 'Аня', 'Виктор').
