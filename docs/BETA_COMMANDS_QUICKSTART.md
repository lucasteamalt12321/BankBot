# 🚀 Быстрый старт: Бета-команды

## ✅ Что реализовано

Все 27 бета-команд теперь доступны в боте!

### 💰 Экономика и торговля (7 команд)
- `/market` - Рыночная площадка
- `/sell` - Выставить на продажу
- `/trade` - Обмен с игроками
- `/loan` - Взять кредит
- `/repay` - Погасить кредит
- `/invest` - Инвестировать
- `/portfolio` - Портфель

### 🎯 Квесты (5 команд)
- `/quests` - Список квестов
- `/quest_start` - Начать квест
- `/quest_progress` - Прогресс
- `/quest_complete` - Завершить
- `/quest_history` - История

### 🏆 Рейтинги (5 команд)
- `/leaderboard` - Таблица лидеров
- `/rank` - Ваш ранг
- `/compare` - Сравнение
- `/tournament` - Турниры
- `/tournament_join` - Участие

### 🎨 Персонализация (4 команды)
- `/customize` - Настройки
- `/title` - Титул
- `/badge` - Значки
- `/theme` - Тема

### 📊 Аналитика (3 команды)
- `/activity` - Активность
- `/income` - Доходы
- `/expenses` - Расходы

### 🎁 Бонусы (3 команды)
- `/events` - События
- `/spin` - Колесо фортуны
- `/lottery` - Лотерея

---

## 📦 Установка

### Шаг 1: Применить миграцию базы данных

```bash
python database/migrations/apply_beta_migration.py
```

Эта команда создаст все необходимые таблицы для бета-функций.

### Шаг 2: Перезапустить бота

```bash
python run_bot.py
```

### Шаг 3: Проверить работу

Отправьте боту команду:
```
/beta
```

Вы должны увидеть список всех доступных бета-команд.

---

## 🎮 Как использовать

### Попробуйте прямо сейчас:

1. **Крутите колесо фортуны:**
   ```
   /spin
   ```
   Получайте случайные награды каждый день!

2. **Проверьте свой ранг:**
   ```
   /rank
   ```
   Узнайте свой уровень и прогресс.

3. **Посмотрите квесты:**
   ```
   /quests
   ```
   Выполняйте задания за награды.

4. **Проверьте события:**
   ```
   /events
   ```
   Узнайте о текущих акциях и бонусах.

5. **Таблица лидеров:**
   ```
   /leaderboard
   ```
   Посмотрите топ игроков.
