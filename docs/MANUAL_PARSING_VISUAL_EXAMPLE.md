# Визуальный пример ручного парсинга

## Сценарий использования

### Шаг 1: Игровой бот отправляет сообщение

```
┌─────────────────────────────────────┐
│ 🤖 Shmalala Bot                     │
├─────────────────────────────────────┤
│ 🎣 [Рыбалка] 🎣                     │
│                                     │
│ Рыбак: @username                    │
│ Монеты: +50 (150)💰                 │
└─────────────────────────────────────┘
```

### Шаг 2: Вы отвечаете на сообщение

```
┌─────────────────────────────────────┐
│ 👤 Вы                               │
├─────────────────────────────────────┤
│ ↪️ Ответ на сообщение Shmalala Bot  │
│                                     │
│ парсинг                             │
└─────────────────────────────────────┘
```

### Шаг 3: Бот обрабатывает и отвечает

```
┌─────────────────────────────────────┐
│ 🤖 LucasTeam Bank Bot               │
├─────────────────────────────────────┤
│ ✅ Сообщение успешно обработано!    │
│                                     │
│ 🎣 username поймал рыбу!            │
│ 💰 Начислено: 50 монет              │
│ 💳 Новый баланс: 200 монет          │
│                                     │
│ 🤖 Бот: Shmalala (@shmalala_bot)   │
└─────────────────────────────────────┘
```

## Диаграмма процесса

```
Игровой бот → Сообщение → Пользователь видит
                ↓
         Пользователь отвечает "парсинг"
                ↓
         Bank Bot получает команду
                ↓
         Проверка: это ответ на сообщение бота?
                ↓
         Парсинг содержимого сообщения
                ↓
         Начисление монет/очков
                ↓
         Отправка подтверждения
```
