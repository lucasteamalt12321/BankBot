# Telegram-бот банк-аггрегатор LucasTeam

## Структура проекта

```
├── bot/                    # Telegram-бот
│   ├── bot.py             # Основной код бота (объединенный)
│   ├── main.py            # Точка входа
│   ├── parsers.py         # Парсеры игровых сообщений
│   └── parser_config.py   # Конфигурация парсеров
├── core/                  # Основная бизнес-логика
├── database/              # Работа с базой данных
├── utils/                 # Утилиты и вспомогательные функции
├── tests/                 # Тесты
├── config/                # Конфигурационные файлы
├── data/                  # Файлы данных (БД, JSON)
├── docs/                  # Документация
└── run_bot.py            # Скрипт запуска бота
```

## Запуск бота

### Способ 1: Через main.py
```bash
python bot/main.py
```

### Способ 2: Через run_bot.py (рекомендуется)
```bash
python run_bot.py
```

## Основные функции

- 🏦 Банковская система с автоматическим начислением монет
- 🎮 Поддержка игр: Shmalala, GD Cards, True Mafia, Bunker RP
- 🛍️ Магазин товаров и привилегий
- 🎯 Мини-игры (города, слова-убийцы, уровни GD)
- 🎲 D&D мастерская
- 🎁 Ежедневные бонусы и задания
- 🏆 Система достижений
- 👥 Социальные функции (друзья, кланы, подарки)
- 📊 Административные команды
- 🔔 Система уведомлений

## Конфигурация

Настройте файл `.env` с токеном бота и другими параметрами.
