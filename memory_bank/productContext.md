# Product Context — BankBot

## Зачем нужен проект

LucasTeam — сообщество игроков, участвующих в нескольких Telegram-играх одновременно. Проблема: у каждой игры своя валюта и нет единого способа отслеживать активность и вознаграждать участников.

BankBot решает это: автоматически парсит результаты игр и начисляет единую валюту (банковские монеты), которую можно тратить в общем магазине.

## Проблемы пользователей

1. Нет единого баланса — нужно следить за несколькими играми отдельно
2. Ручное начисление очков — администраторы тратят время на подсчёт
3. Нет мотивации — отсутствует система наград и достижений
4. Нет социальных функций — игроки не могут взаимодействовать между играми

## Целевая аудитория

Участники Telegram-сообщества LucasTeam, играющие в:
- GD Cards (карточная игра)
- Shmalala (рыбалка)
- True Mafia (мафия)
- Bunker RP (ролевая игра)

## Ключевые сценарии использования

1. Игрок получает монеты автоматически после игры (парсинг)
2. Игрок проверяет баланс командой `/balance`
3. Игрок тратит монеты в магазине (`/shop`, `/buy`)
4. Администратор начисляет/списывает монеты вручную
5. Администратор делает рассылку всем пользователям

## Метрики успеха

- 22+ активных пользователя
- 50+ команд бота
- Автоматический парсинг 4 игр
- Время ответа бота < 2 сек
